Constructor Menciptakan cell sesuai dengan parameter. Semua habitat tidak bisa diakses (not is_accessible). Road dan semua habitat memiliki nama "" (string kosong). Parameter _entrance dan _exit hanya boleh true saat _type adalah Road.

Parameters
_accessibleNilai true/false cell dapat diakses. Nilai default = false.
_typeTipe cell. Nilai default = "".
_nameNama fasilitas cell. Nilai default = "".
_entranceNilai true/false cell merupakan entrance. Nilai default = false.
_exitNilai true/false cell merupakan exit. Nilai default = false.
